Which row breaks first?

The dessert name row break first.

Explaination:

The dessert names (such as "Strawberry Shortcake") are longer in length compared to the individual letters in my last name. As the browser window is resized and becomes narrower, the longer text in the dessert name row takes up more space and therefore breaks earlier. The letters in my last name, being much shorter, are able to fit in the shrinking window longer, causing the dessert row to break before the letter row.
